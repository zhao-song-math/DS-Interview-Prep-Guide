# Recursion 递归

![递归流程图](pics/feb6ed14c24b886de5421eeb5f15ae3f.png) [^1]

**Bottom up recursion 3 steps:**[^2]
1. Ask for subproblem result
2. Do something in current level of recursion
3. Return result

❗️注意：**递归必须要有一个退出的条件！**
